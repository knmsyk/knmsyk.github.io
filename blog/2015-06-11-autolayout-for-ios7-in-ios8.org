iOS8 では問題なくレイアウトされていたけど、iOS7 で動かしたら下記のエラーでクラッシュした。
"NSInternalInconsistencyException Auto Layout still required after sending -viewDidLayoutSubviews to the view controller"

対策は、viewDidLayoutSubviews の最後に下記を追加する。
``` objc
[self.view layoutIfNeeded];
```

もし UILabel のレイアウトを先に更新しておきたい場合は、下記をさきによんでおく。
``` objc
[label layoutIfNeeded];
```
