*** Return and jumps
***** Break and Continue Labels
- Labeling

``` kotlin
loop@ for (i in 1..100) {
  for (j in 1..100) {
    if (...)
      break@loop
  }
}
```

- Automatic labeling
``` kotlin
fun foo() {
  ints.forEach {
    if (it == 0) return@forEach
    print(it)
  }
}
```
``` kotlin
fun outer() {
  fun inner() {
    return@outer // the label @outer was defined automatically
  }
}
```
