1. Set up [[https://github.com/yoshinari-nomura/org-octopress][org-octopress]]
2. Write your post with 'org-octopress-new-post' command
3. Command with 'C-c C-e P x' and type 'octopress' for project to export a HTML file
4. Go to octopress root directory and 'rake gen_deploy'
