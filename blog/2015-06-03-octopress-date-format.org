date format の設定方法と Post 一覧で時刻を消して日付のみを表示する方法について。

{{{more}}}

** _config.yml に設定を追加する
``` yaml
#date_format: "ordinal"
date_format: "%Y/%m/%d (%a)"
time_format: "%H:%M"
```

** Post 一覧で時刻を消して日付のみを表示する
- source/_includes/post/date.html 変更前
```
{% if post.date %}{% capture time %}{{ post.date_time_html }}{% endcapture %}{% endif %}
```

- source/_includes/post/date.html 変更後
```
{% if post.date %}{% capture time %}{{ post.date_html }}{% endcapture %}{% endif %}
```
