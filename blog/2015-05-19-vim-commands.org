* Move
** Cursor
| w         | 次の単語に移動                                                                   |
| b         | 前の単語に移動                                                                   |
| e         | 単語の末尾に移動                                                                 |
| W         | 次の単語に移動（記号などは無視）                                                 |
| B         | 前の単語に移動（記号などは無視）                                                 |
| E         | 単語の末尾に移動（記号などは無視）                                               |
| ^         | 行頭に移動                                                                       |
| $         | 行末に移動                                                                       |
| )         | 次の文の先頭へ移動（文は、.!?+スペースで区切られた固まり）                       |
| (         | 前の文の先頭へ移動                                                               |
| }         | 次の段落の先頭へ移動（段落は、空行で終わっている固まり）                         |
| {         | 前の段落の先頭へ移動                                                             |
| H         | ウインドウ内の上に移動                                                           |
| M         | ウインドウ内の中に移動                                                           |
| L         | ウインドウ内の下に移動                                                           |
| gg        | ファイルの先頭へ ( 1G より便利 )                                                 |
| G         | ファイルの末尾へ ( 0G より便利 )                                                 |
| %         | 対応する括弧へ移動                                                               |
| *         | カーソル位置のある単語の次の単語へ ( var にカーソルがあったら、次の var に飛ぶ ) |
| #         | カーソル位置のある単語の前の単語へ                                               |
| f[string] | その行の [string] へ ( / を打つより便利 )                                        |
| t[string] | その行にある [string] の前へ                                                     |

** Window
| C-f      | 1画面ずつ下へ                                                  |
| C-b      | 1画面ずつ上へ                                                  |
| C-d      | 半画面ずつ下へ                                                 |
| C-u      | 半画面ずつ上へ                                                 |
| C-e      | 1行ずつ下へ                                                    |
| C-y      | 1行ずつ上へ                                                    |
| z<enter> | 現在カーソルのある行が画面の一番上になるように画面がスクロール |
| z.       | 現在カーソルのある行が画面の中央になるように画面がスクロール   |
| z-       | 現在カーソルのある行が画面の一番下になるように画面がスクロール |

* Edit
| i           | 挿入モード（カーソル位置から）                                                |            |
| a           | 挿入モード（カーソルの後から）                                                |            |
| I           | 行頭の最初の文字にカーソルを移動してからその直前にテキストを入力する          |            |
| A           | 行末にカーソルを移動してからその直後にテキストを入力する                      |            |
| o           | カレント行の下に空白行を挿入し、その行でテキストを入力する                    |            |
| O           | カレント行の上に空白行を挿入し、その行でテキストを入力する                    |            |
| s           | １文字を削除して挿入モードへ                                                  |            |
| S           | 現在の行を削除して挿入モードへ                                                |            |
| cc          | 「dd」と同じ動作（行を削除）をしてからインサートモードになる。（「S」と同じ） | 2005-03-23 |
| c[移動操作] | 「d」と同じ動作（削除）をしてからインサートモードになる。                     |            |
| J           | 次の行と現在の行とを結合する                                                  |            |
| [数字]J     | 下の入力した数字分の行を現在の行と結合する                                    |            |

* Esc
| C-c | ESC や C-[ と同じ。打ちやすい |
| C-[ | ESC                           |

* Delete
| x           | 1文字を削除（Deleteキーと同じ方向）                              |
| X           | 1文字を削除（BSキーと同じ方向）                                  |
| D           | カーソル位置から行末まで削除                                     |
| d[移動操作] | 現在のカーソル位置と、移動操作によって移動した先の間を削除する。 |
| J           | 次の行を、現在カーソルのある行の末尾に追加（行末文字の削除)      |

* Cut, Yank(Copy), and Paste
| dd | 行をカット               |
| yy | 行をコピー               |
| P  | ペースト（カレント行）   |
| p  | ペースト（カーソルの下） |

* Undo and Redo
| u   | 取り消し（Undo）                     |
| U   | 行に対して行った変更の全てを取り消す |
| C-r | やり直し（:redoと同じ）              |

* Select
| v   | 文字単位のビジュアルモードを開始する。      |
| V   | 行単位のビジュアルモードを開始する。        |
| o   | 選択中カーソルを逆の端点へ移動              |
| O   | 矩形選択時、「o」では移動できない端点へ移動 |
| C-v | 矩形                                        |

* Window
| :sp        | 画面を上下に分割（:splitと同じ）                                  |
| :vsp       | 画面を左右に分割（:vsplitと同じ）                                 |
| :q         | ウィンドウを閉じる                                                |
| :qall      | 全てのウィンドウを閉じる                                          |
| :only      | カレントウィンドウ以外を閉じる                                    |
| C-wo       | カレントウィンドウ以外を閉じる                                    |
| C-ww       | ウィンドウを順番に移動                                            |
| C-w[hjkl]  | ウインドウを{上,下,左,右}に移動                                   |
| C-w[HJKL]  | カレントウィンドウを一番{上,下,左,右}に移動、（分割縦横切り替え） |
| [数値]C-w+ | ウィンドウサイズを数字分増やす                                    |
| [数値]C-w- | ウィンドウサイズを数字分減らす                                    |

* Buffer
| :ls | 現在開いているバッファを確認する |

* Search
| /[searching word]<return> | 現在のカーソル位置から下方向に検索する |
| ?[searching word]<return> | 現在のカーソル位置から上方向に検索する |
| n                         | 直前の検索を繰り返す                   |
| N                         | 直前の検索を逆方向に繰り返す           |

* Replace
| R                                        | 以降、上書きしながら編集（Replaceモード）                          |
| r                                        | カーソル上の文字を置換                                             |
| :%s/[searching word]/[replacing word]    | もっとも近くにある文字列一つに対してのみ実行                       |
| :%s/[searching word]/[replacing word]/g  | ファイルの中すべての文字列に対して、置換えを行う                   |
| :%s/[searching word]/[replacing word]/gc | ファイルの中すべての文字列に対して、置換えを行う。逐次問い合わせ。 |

* Open
| :e                  | ファイルを開く。TAB 補完や *.html とか、入力補助が便利                   |
| :bdelete            | ファイルを閉じる                                                         |
| :e **/Root.pm       | ディレクトリを指定しなくても Root.pm を探し出して開いてくれる            |
| :e **/*Ro           | Root.pm にマッチすれば開いてくれる。                                     |
| :e ../**/.bashrc    | 上の階層も                                                               |
| gf                  | カーソル位置にある単語をファイル名と見なして、そのファイルを開いてくれる |
| :echo expand("%:p") | 現在開いているファイルのフルパスを表示                                   |
| :ls                 | 開いているファイルの一覧                                                 |
| :bn                 | 次のファイルへ                                                           |
| :bp                 | 前のファイルへ                                                           |
| :b#                 | 直前のファイル                                                           |
| :b ファイル名       | すでに開いているファイルを開く                                           |

* Save
| :wq | write and quit |
| ZZ  | same as :wq    |
| q!  | force quit     |
